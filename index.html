Команды для работы с консолью
[ D: ] переход на другой диск
[ cd Git ] переход в нижестоящий каталог
[ сd .. ] переход на каталог, находящийся выше
[ cd \ ] переход в корневой каталог
[ dir ] просмотр содержимого каталога
Команды для работы с Git
[ git init ] создание репозитория в директории
[ git clone ] клонировать удаленный репозиторий в локальны
[ git status ] проверка наличия изменений в локальном репозитории
[ git add . ] добавление в ИНДЕКС всех внесенных изменений без исключений
[ git add Learning_GitHub/src/SB_02.java ] добавление в ИНДЕКС конкретных измененных файлов
